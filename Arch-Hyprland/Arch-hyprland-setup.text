10.Making waybar autoreload with script:

- To reload waybar after pywal gives it new css file we need a app:
#  sudo pacman -S inotify-tools

- Make a bash script to do this:

-----------------------------------------------------------------------------
#!/bin/bash
COLORS="$HOME/.cache/wal/colors-waybar.css"
inotifywait -m -e close_write "$COLORS" | while read -r; do
    pkill -SIGUSR2 waybar
done
-----------------------------------------------------------------------------

- Make it exicutable and make it autorun in your hyprland.
- Note:
‚úÖ pkill -SIGUSR2 waybar does NOT kill Waybar ‚Äî it sends a signal (SIGUSR2) that Waybar understands as a "reload" command, if it's compiled with that support (which most versions from the Arch repo are).
‚ùå killall waybar does kill it, and then you restart it manually.
