ðŸŸ¢ 2. To make pc screen off after a period of time use swayidle:

- Install it:
#  sudo pscman -S swayidle

- In hypr config add this:

exec-once = swayidle -w \
  timeout 300 '~/.config/scripts-arch/lock.sh' \
  timeout 330 'hyprctl dispatch dpms off' \
  resume 'hyprctl dispatch dpms on' \
  before-sleep '~/.config/scripts-arch/lock.sh'


-timeout 300 '~/.config/scripts-arch/lock.sh': Locks the screen after 5 minutes of inactivity.
-timeout 330 'hyprctl dispatch dpms off': Turns off the screen 30 seconds after locking.
-resume 'hyprctl dispatch dpms on': Turns the screen back on when resuming.
-before-sleep '~/.config/scripts-arch/lock.sh': Locks the screen before suspending (triggered by lid close or systemctl suspend).
